# .files

Contains my dotfiles that I sync via [Stow](https://www.gnu.org/software/stow/)

## Installation

* Install [Stow](https://www.gnu.org/software/stow/) if you haven't already
* Clone this repo
```bash
git clone https://github.com/silverAndroid/.files ~/dotfiles
```

## Usage

```bash
cd ~/dotfiles
stow .
```
